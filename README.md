# Amazon Growth Studio Website

## Overview
A professional corporate website for Amazon Growth Studio - an Amazon FBA service agency.

## Pages
1. **Home** (index.html) - Hero section, stats, services preview, and CTA
2. **About** (about.html) - Company story, mission, vision, and core values
3. **Services** (services.html) - Detailed service offerings and process
4. **Contact** (contact.html) - Contact information and business hours
5. **Blog** (https://blog.amazongrowthstudio.online) - Blog with wordpress installed on subdomain

## Technologies Used
- **HTML5** - Semantic markup
- **Tailwind CSS** (via CDN) - Utility-first styling
- **Vanilla JavaScript** - Interactive features
- **Font Awesome** - Icons
- **Google Fonts** - Oswald (headings) and Manrope (body)

## Features
- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll animations
- Mobile-friendly navigation
- Professional color scheme (Obsidian, Amazon Gold, Electric Blue)
- Accessibility-focused with data-testid attributes
- Counter animations for statistics
- Hover effects and transitions

## Color Palette
- **Primary**: #0F172A (Obsidian)
- **Accent**: #FF9900 (Amazon Gold)
- **Secondary**: #007AFF (Electric Blue)
- **Background**: #F8FAFC (Paper White)

## How to Use
1. Open `index.html` in a web browser
2. Navigate between pages using the navigation menu
3. All files are static and require no build process

## File Structure
```
/website
├── index.html       # Home page
├── about.html       # About page
├── services.html    # Services page
├── contact.html     # Contact page
-- /blog/ 		#Blog
├── styles.css       # Custom CSS animations and styles
├── script.js        # JavaScript for interactivity
└── README.md        # This file
```

## Customization
To customize the website:
1. Update contact information in `contact.html`
2. Modify colors in the Tailwind config section of each HTML file
3. Add/remove services in `services.html`
4. Update company information in `about.html`

## Browser Support
Supports all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
